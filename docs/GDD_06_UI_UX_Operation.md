# GDD-06: UI/UX & Operation

## Project Title: HISTORY OF SOVEREIGNS

### 6.1 UI 구조 (Mobile-First)

- **Layout:** Mobile(단일 뷰) / Desktop(3단 그리드).
- **Tabs:** Dashboard, Map, Ranking, **Admin(조건부 노출)**.

### 6.2 국가 생성 (One-Time Form)

- **단계:** 8개 질문 입력 -> [건국] 버튼.
- **주의사항:** "생성 후에는 되돌릴 수 없습니다" 경고 모달 띄움.
- **처리:** 유저가 입력한 이름을 그대로 사용하며, 리롤 기능 없음.

### 6.3 관리자 대시보드 (Admin Panel)

관리자 UID로 로그인했을 때만 접근 가능한 시크릿 페이지(`/admin`)입니다.

1.  **User Management:**
    - 전체 유저 리스트 조회 (검색 기능).
    - 특정 유저 상세 정보 조회 및 **스탯/자원 강제 수정**.
    - 악성 유저 차단(Ban) 및 국가 삭제.
2.  **System Control:**
    - 긴급 점검 모드 ON/OFF.
    - 글로벌 뉴스(속보) 직접 작성 및 송출.
    - AI 프롬프트 테스트 (비용 없이 테스트).

### 6.4 시각 효과

- **Map:** Canvas 기반 10,000 타일 렌더링.
- **Visuals:** 빈곤/전쟁 상태에 따른 화면 필터 효과.

### 6.5 핵심 화면 플로우

- **온보딩:** 세계관 설명 3줄 → 질문지 입력 → 국가 포스터(이름/태그/색상) 생성 → 세계 뉴스 데뷔 카드.
- **대시보드:** 국가 전반 요약, 쿨다운 타이머, 자원/스탯 바, 지난 결정 3개 로그, 다음 제안(추천 행동) 제공.
- **행동 작성기:** 자연어 입력 + 전술/정책 템플릿 버튼. 리스크/예상 쿨타임/자원 소모를 미리 표시.
- **전쟁/보고서 뷰어:** 전선별 로그, 맵 미니뷰, 전리품/피해 요약, '영웅 결정' 하이라이트.
- **부재중 보고서:** 오프라인 피해 요약, 대응 바로가기(복구/반격) 버튼.

### 6.6 지도(Map) UX

- 10,000 타일을 줌/패닝. 모바일은 드래그, 데스크톱은 WASD/스크롤 지원.
- 소유 국가마다 국기/색상 오버레이, 중요 이벤트는 아이콘으로 표기.
- 전쟁 중인 타일은 애니메이션(불길/연기), 보호막은 돔 이펙트.
- 타일 클릭 → 타일 정보, 소유국, 최근 이벤트, 이동/공격/정찰 CTA.

### 6.7 피드백/알림

- **즉각 피드백:** 액션 제출 즉시 '서약' 배너, 쿨다운 타이머 시작, 예상 판정 바 노출.
- **알림 채널:** 인앱 배지, 뉴스 티커, 웹 푸시(옵트인), 이메일(멸망/월간 리포트).
- **감정 연출:** 승리 시 황금색 라이트, 패배 시 화면 흔들림+회색 필터, 멸망 시 느린 줌아웃과 추모음.

### 6.8 랭킹/커뮤니티

- **랭킹:** 국력, 지속일, 전쟁 승률, 문화 영향력 등 다중 탭. 매일 리셋되는 데일리 챌린지 랭킹 추가.
- **뉴스 티커:** AI가 생성한 세계 소식 스트립. 국가 이름을 반복 노출해 존재감 강화.
- **기념비:** 멸망 국가의 추모 카드(상징 문장/최후의 결정)를 맵에 배치.

### 6.9 접근성/편의

- 색약 모드(맵 팔레트 교체), 폰트 크기 확대, 고대비 모드.
- 느린 네트워크 모드: 애니메이션 축소, 텍스처 해상도 감소.
- 입력 도움: 전술/정책 추천 버튼, 최근 명령 자동 완성.

### 6.10 운영 UI

- **패치 노트 섹션:** 공지/핫픽스 배너 상시 노출, 변경된 밸런스 계수 표기.
- **서버 상태:** 점검 시 카운트다운과 대체 활동 제안(기록 읽기/스토리 모드).
- **CS 지원:** 신고/의견 제출 폼, 관리자 응답 로그 표시.
