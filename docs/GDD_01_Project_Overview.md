# GDD-01: Project Overview & User Experience

## Project Title: HISTORY OF SOVEREIGNS

### 1.1 프로젝트 정의

- **타이틀:** HISTORY OF SOVEREIGNS (히스토리 오브 소버린)
- **장르:** AI Narrative Grand Strategy (AI 대전략 시뮬레이션)
- **플랫폼:** **Mobile Web (Primary)** & Desktop Web
- **기술 스택:** Next.js, Firebase, CSS Modules, Google Gemini 2.5 Flash-Lite
- **디자인 원칙:** Mobile-First Responsive.

### 1.2 기획 의도 (Vision)

플레이어는 전지전능한 통치자가 되어 **자연어(Natural Language)**로 명령을 내립니다.
모든 결정은 되돌릴 수 없으며(No Reroll), 한 번 건국한 국가는 멸망할 때까지 운명을 함께해야 합니다.

### 1.3 핵심 특징 (Key Features)

1.  **One Life, One Nation (단일 국가 원칙):**
    - 유저는 오직 하나의 국가만 운영할 수 있습니다.
    - 생성 완료 시 수정이나 '리롤(Reroll)'은 불가능합니다. 멸망해야만 새로 시작할 수 있습니다.
2.  **User-Defined Genesis (유저 주도 건국):**
    - 국가 이름은 유저가 직접 짓습니다. AI는 유저의 설정을 분석하여 스탯을 부여할 뿐입니다.
3.  **Live Hex Map & War:**
    - 실시간으로 변하는 10,000개의 타일 지도와 동맹/흡수 시스템.
4.  **Server Authority:**
    - 유저가 탭을 닫아도 서버 로직은 끝까지 수행되어 DB에 기록됩니다.
5.  **Admin System (관리자 모드):**
    - 특정 권한을 가진 관리자만 접근 가능한 대시보드를 통해 유저 관리, 공지 발송, 데이터 조작 등을 수행합니다.

### 1.4 플레이어 경험 목표

- **자유도:** 자연어 명령으로 외교/전쟁/내정/과학/문화까지 폭넓은 선택을 허용. 같은 입력이라도 세계 상황과 문화 태그에 따라 다른 결과를 제공.
- **강한 책임감:** 되돌릴 수 없음, 영구 멸망, 전 세계 뉴스 노출로 결정의 무게를 느끼게 함.
- **즉각적 피드백:** 3분 쿨다운 안에서도 자원 흐름, 뉴스 티커, 이벤트 로그로 계속 움직임을 보여줌.
- **서사 몰입:** AI가 국가의 정체성, 문화, 전쟁 보고서를 일관된 톤으로 생성. 멸망 시 추모 페이지 제공.
- **공정성:** 약탈 상한, 보호막, 부재중 방어 보고서로 박탈감 최소화.

### 1.5 타깃 오디언스 & 플레이 패턴

- **타깃:** 서사형 전략을 좋아하고 짧은 세션(5~10분)으로도 성취감을 원하는 20~40대.
- **플레이 패턴:** 출근/등교 전 1회, 점심 1회, 저녁 1~2회 접속. 세션마다 한두 개 핵심 결정을 내리고, 공지/뉴스/랭킹을 읽음.
- **온보딩:** 8문항 건국 보고서 -> AI가 국가 포스터/헤드라인 생성 -> 세계 뉴스에 데뷔.

### 1.6 컨텐츠 방향성

- **초기 72시간 챕터 구조:** 국가 탄생, 첫 외교, 첫 전쟁, 기술 분기점, 내부 소요/혁명 이벤트를 시나리오처럼 배치.
- **월간 시즌 테마:** 기후 재난, 초지능 AI 의회, 대항해, 자원 붕괴 등 테마별로 세계 규칙 일부를 바꿔 반복 재미 제공.
- **커뮤니티 감정선:** 월별 '역사 시상식'( longest standing, best storyteller, tragic hero )를 뉴스로 송출.

### 1.7 유저 만족/자유도 설계 원칙

- **입력 자유도:** 필수 필드는 최소화, 서술형 응답은 제한 없는 자유 서술. AI는 과도한 검열 없이 톤만 정제.
- **정책 다양성:** 경제·정치·군사·사상·기술 정책을 병렬로 실행 가능하되, 쿨타임은 공유.
- **실패 서사 보상:** 패배/멸망 시 히스토리얼 카드(국가 연표, 최후의 헤드라인, 대표 결정 TOP3)로 여운을 제공.
- **존재감:** 뉴스 티커, 월드 맵 배너, 랭킹에서는 짧은 수식어를 붙여 국가를 반복 노출.

### 1.8 성공 지표 (라이브 KPI)

- D1/D7/D30 Retention, 평균 세션 길이(목표 6~8분), 일별 주요 액션 수(2~4회).
- 멸망 후 재시작률, 멸망 후 추모 페이지 공유율.
- 보호막 이후 재로그인율, 전쟁 보고서 읽기율, 관리자 공지 열람율.

### 1.9 라이브 운영 루프

- **일일:** 전쟁/이벤트 로그 요약 푸시(웹 푸시/이메일), 뉴스 티커 3~5건 발행.
- **주간:** 시즌 룰 패치 노트, 버프/너프 발표, 커뮤니티 투표 결과 반영.
- **월간:** 세계 리셋 없이 계층적 시즌(새 규칙 추가) 진행. 중요 패배 국가의 '기념비'를 맵에 남김.
