# GDD-03: Nation System & Stats

## Project Title: HISTORY OF SOVEREIGNS

### 3.1 국가 생성 프로세스 (The Founding Form)

AI와의 대화가 아닌, **8개의 질문**에 대한 답변(보고서) 제출 방식입니다.
**중요:** 제출 즉시 국가가 생성되며, 되돌리거나 다시 뽑기(Reroll)는 없습니다.

#### [건국 보고서 질문지]

1.  **국가명 (Name):** (필수, 유저 직접 입력)
2.  **지도자 칭호:** (예: 황제, 대족장)
3.  **기후 및 지형:** [Dropdown] (사막, 설원, 정글, 지하 등)
4.  **정치 체제:** [Dropdown] (군주정, 민주정, AI 통제 등)
5.  **경제 중점:** [Dropdown] (농업, 공업, 군수 등)
6.  **사회적 분위기:** (50자 서술)
7.  **국가의 기원:** (100자 서술)
8.  **치명적 약점:** (50자 서술)

### 3.2 스탯 배분 로직 (질문지 기반)

- **직접 매핑:** 정치 체제/경제 중점/기후는 고정 가중치로 `Stability`, `Economy`, `Sustainability` 초기값을 결정.
- **서술 분석:** "사회적 분위기", "국가의 기원", "치명적 약점" 서술을 키워드 매핑해 `Happiness`, `Influence`, `Technology` 보정치 부여.
- **약점 보정:** 치명적 약점이 강할수록 `Stability`를 낮추되 다른 한 스탯에 +5~10 보너스를 주어 위험/잠재력을 동시 부여.
- **초기 밸런스:** 총합 350±10에서 정규화. 극단 편향은 허용하되 부적절 콘텐츠만 필터.

### 3.3 확장된 국력 지표 (7 Stats)

1.  **Stability (안정도):** 0 도달 시 멸망 위기.
2.  **Economy (경제력)**
3.  **Military (군사력)**
4.  **Happiness (행복도)**
5.  **Technology (기술력)**
6.  **Sustainability (지속가능성):** 환경/자원 유지력.
7.  **Influence (영향력):** 외교력.

### 3.4 4대 자원 (Resources)

1.  **Gold (국고)**
2.  **Food (식량)**
3.  **Materials (자재):** 건설/제작용.
4.  **Energy (에너지):** 유지/가동용.

### 3.5 환경 및 속성

- **Climate Effect:** 기후에 따른 자원 생산/소모 보정값 적용.
- **Culture Tags:** AI가 생성 시 부여하는 3~5개의 태그 (자동 생성).

### 3.6 인구·영토·도시

- **Population:** 식량/안정도/행복도로 성장. 기근 시 감소, 신규 도시 건설 시 분산.
- **Territory:** 기본 9타일(3x3) 시작. 전쟁/개척으로 확장. 10,000 타일 세계에서 타일마다 자원/지형 상이.
- **City Slots:** 초기 1개, 기술/경제 정책으로 최대 5개까지 확장. 도시마다 특화(군수, 상업, 연구, 문화) 선택.

### 3.7 특성, 문화 태그, 이념

- **Culture Tags:** 예) "전사", "해양", "학자", "신정", "노마드", "AI 의회" 등. 태그는 이벤트 톤과 전쟁/정책 결과에 영향.
- **National Traits:** AI가 질문지 기반 2개 생성(예: "무역 집착", "영광 우선"). 플레이어는 1개를 잠금 해제해 즉시 적용.
- **Ideology Drift:** 외교/정책/문화 이벤트에 따라 정치 체제가 변동(왕정→입헌→AI 의회). 변화는 장단점과 반란 위험을 동반.

### 3.8 행복도/안정도 세부 규칙

- **행복도 상승:** 복지/문화 투자, 승리 보상, 안정된 식량, 스토리 이벤트 성공.
- **행복도 감소:** 장기 전쟁 피해, 세금/징병, 이벤트 실패, 금수조치.
- **안정도 영향:** 행복도 장기 저하는 안정도 지속 하락. 안정도 0 도달 시 "붕괴 타이머"(24시간) 발동, 반전 실패 시 멸망.
- **붕괴 역전:** 반란 진압, 지원금, 사상 전환으로 붕괴 타이머를 상쇄. 실패 시 국가 멸망과 함께 추모 연표 생성.

### 3.9 성장 및 진화 루프

- **정책 슬롯:** 동시에 3개 정책 유지. 일부는 지속효과, 일부는 단발성. 정책 변경에는 쿨타임 공유와 안정도 비용이 있음.
- **기술 분기:** 군사/산업/과학/문화/환경 5계열에서 투자. 특정 조합은 레전더리 이벤트를 언락.
- **레전더리 이벤트:** 태그+기술+전쟁 승리 조합 시 고유 사건 발동(예: "해류를 제어한 왕국", "AI 의회가 반란을 제압"). 국가 서사의 하이라이트로 기록.
