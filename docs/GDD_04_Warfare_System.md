# GDD-04: Warfare & Interactions

## Project Title: HISTORY OF SOVEREIGNS

### 4.1 전쟁 프로세스 (Narrative War)

1.  **공격 명령:** 전술 입력 (자연어).
2.  **AI 판정:** 국력 + 전술 + 지형 + 기술력 종합 판정.
3.  **결과:** 승리 시 자원 약탈 및 타일 강탈.

#### 전쟁 페이즈

- **개전:** 선언문/명분 생성, 사기 계산, 지원 세력 체크(동맹 여부).
- **교전:** 전선당 주사위가 아닌 가중치 전투 계산(지형 보정+기술+사기). 기습/포위/해상차단 등 전술 옵션을 AI가 서술로 풀어냄.
- **결과:** 전리품, 사상자, 지형 변화, 사기 변화, 전쟁 피로도 축적. 보고서가 뉴스/알림/로그로 발송.

### 4.2 흡수와 합병 (Annexation)

- **조건:** 적의 `Stability`가 0인 상태에서 수도 타일 점령.
- **결과:** 패배한 국가는 **영구 삭제**되며, 모든 자원/영토가 승자에게 흡수됨.
- **알림:** 전 세계 뉴스 티커에 멸망 소식 전송.

### 4.3 안전 장치

- **Loot Cap:** 자원 약탈 30% 제한.
- **Shield:** 패배 시 4~8시간 보호막.

### 4.4 관리자 개입 (Admin Intervention)

- 관리자는 분쟁 발생 시 특정 전쟁을 강제 중단시키거나, 특정 국가에 '평화 유지군(시스템 보호막)'을 강제로 씌울 수 있음.

### 4.5 외교, 무역, 영향력

- **조약 유형:** 동맹, 불가침, 상호방위, 조공, 공동 연구, 문화 교류. 조약마다 유지비/쿨타임/파기 패널티 존재.
- **무역:** 식량/에너지/자재/골드 거래. 거래량은 운송력(기술/항만/도로)으로 제한. 사기성 거래 방지를 위해 양측 상호 확인 UI + 15초 쿨링.
- **영향력 소모:** 외교 행동은 `Influence`를 소비하고 성공 시 보충. 영향력이 0이면 제안이 묵살될 확률 증가.

### 4.6 전쟁 피로도와 사기

- **War Exhaustion:** 전투 횟수, 사상자, 점령 실패로 누적. 높을수록 행복도/안정도 하락, 반란 이벤트 확률 증가.
- **Morale:** 초기엔 애국심/명분으로 상승, 장기전/패배/기근으로 감소. 사기가 낮으면 전투 가중치 감소, 도주/탈영 서술이 보고서에 포함.
- **사기 회복:** 휴전, 보급, 문화 이벤트, 지도자 연설.

### 4.7 지형·기술 특수 룰

- **지형:** 설원(보급 소모↑), 정글(정찰 제한), 산악(방어 보너스), 해양(해군 필요), 지하(특수 전투).
- **기술:** 공성, 공중, 사이버, AI 참모, 재생 에너지, 기후 제어 등. 특정 기술은 맵 타일 상태까지 변화(습지→농경지).
- **특수 전술:** 포위(식량 소모를 강제), 해상 차단(무역 차단), 공중 폭격(사기 타격 크고 약탈 적음), 사이버 공격(에너지/영향력 공격).

### 4.8 첩보/파괴공작

- **정찰:** 상대 stats, 자원, 보호막, pending_events 일부 공개.
- **교란:** 식민지 부추기기, 허위 뉴스 삽입, 생산 방해. 실패 시 발신국 노출 및 영향력/안정도 손실.
- **방어:** 방첩 정책, 암호화 기술 연구로 첩보 성공률 감소.

### 4.9 플레이어 서사 강화를 위한 연출

- 전투 보고서는 승패 외에도 '가장 빛난 결정', '치명적 실수', '역사적 한 줄'을 포함.
- 멸망 시 해당 전쟁 로그는 '최후의 전투'로 고정되어 모두 열람 가능.
- 동맹이면 공동 연설, 공동 승전 퍼레이드 같은 서사 이벤트가 발생해 만족감을 높임.
